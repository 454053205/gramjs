DiscardCallRequest

Only users can use this method. See code examples.

---functions---
phone.discardCall#b2cbc1c0 flags:# video:flags.0?true peer:InputPhoneCall duration:int reason:PhoneCallDiscardReason connection_id:long = Updates

Returns

Updates

This type can be an instance of either:

UpdatesUpdatesCombined
UpdateShortUpdateShortChatMessage
UpdateShortMessageUpdateShortSentMessage
UpdatesTooLong

Parameters

videoflagThis argument defaults to null and can be omitted.
peerInputPhoneCall
durationint
reasonPhoneCallDiscardReason
connection_idlong

Known RPC errors

This request can cause 2 known errors:

CallAlreadyAcceptedErrorThe call was already accepted.
CallPeerInvalidErrorThe provided call peer object is invalid.

You can import these from gramjs/errors.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.phone.DiscardCallRequest(
        peer=new types.InputPhoneCall(
            id=-12398745604826,
            access_hash=-12398745604826
        ),
        duration=42,
        reason=new types.PhoneCallDiscardReasonMissed(),
        connection_id=-12398745604826
    ));
    console.log(result.stringify());
})();
