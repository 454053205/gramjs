UpdateProfilePhotoRequest

Only users can use this method. See code examples.

---functions---
photos.updateProfilePhoto#f0bb5152 id:InputPhoto = UserProfilePhoto

Returns

UserProfilePhoto

This type can be an instance of either:

UserProfilePhotoUserProfilePhotoEmpty

Parameters

idInputPhoto

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.photos.UpdateProfilePhotoRequest(
        id=new types.InputPhoto(
            id=-12398745604826,
            access_hash=-12398745604826,
            file_reference=b'arbitrary\x7f data \xfa here'
        )
    ));
    console.log(result.stringify());
})();
