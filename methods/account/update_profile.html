UpdateProfileRequest

Only users can use this method. See code examples.

---functions---
account.updateProfile#78515775 flags:# first_name:flags.0?string last_name:flags.1?string about:flags.2?string = User

Returns

User

This type can be an instance of either:

UserUserEmpty

Parameters

first_namestringThis argument defaults to null and can be omitted.
last_namestringThis argument defaults to null and can be omitted.
aboutstringThis argument defaults to null and can be omitted.

Known RPC errors

This request can cause 2 known errors:

AboutTooLongErrorThe provided bio is too long.
FirstNameInvalidErrorThe first name is invalid.

You can import these from gramjs/errors.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.account.UpdateProfileRequest());
    console.log(result.stringify());
})();
