ConfirmPhoneRequest

Only users can use this method. See code examples.

Known RPC errors

This request can cause 2 known errors:

CodeHashInvalidErrorCode hash invalid.
PhoneCodeEmptyErrorThe phone code is missing.

You can import these from gramjs/errors.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.account.ConfirmPhoneRequest(
        phone_code_hash='some string here',
        phone_code='some string here'
    ));
    console.log(result);
})();
