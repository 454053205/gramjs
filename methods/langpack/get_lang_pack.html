GetLangPackRequest

Only users can use this method. See code examples.

---functions---
langpack.getLangPack#f2f2330a lang_pack:string lang_code:string = LangPackDifference

Returns

LangPackDifference

This type can only be an instance of:

LangPackDifference

Parameters

lang_packstring
lang_codestring

Known RPC errors

This request can cause 1 known error:

LangPackInvalidErrorThe provided language pack is invalid.

You can import these from gramjs/errors.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.langpack.GetLangPackRequest(
        lang_pack='',
        lang_code='en'
    ));
    console.log(result.stringify());
})();
