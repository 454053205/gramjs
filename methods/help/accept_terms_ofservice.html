AcceptTermsOfServiceRequest

Only users can use this method. See code examples.

---functions---
help.acceptTermsOfService#ee72f79a id:DataJSON = Bool

Returns

Bool

This type has no instances available.

Parameters

idDataJSON

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.help.AcceptTermsOfServiceRequest(
        id=new types.DataJSON(
            data='some string here'
        )
    ));
    console.log(result);
})();
