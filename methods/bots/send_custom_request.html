SendCustomRequestRequest

Only bots can use this method. See code examples.

---functions---
bots.sendCustomRequest#aa2769ed custom_method:string params:DataJSON = DataJSON

Returns

DataJSON

This type can only be an instance of:

DataJSON

Parameters

custom_methodstring
paramsDataJSON

Known RPC errors

This request can cause 1 known error:

UserBotInvalidErrorThis method can only be called by a bot.

You can import these from gramjs/errors.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.bots.SendCustomRequestRequest(
        custom_method='some string here',
        params=new types.DataJSON(
            data='some string here'
        )
    ));
    console.log(result.stringify());
})();
